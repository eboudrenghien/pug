    Qu'est-ce que c'est ?

    Le templating est plus connu sous le nom de "Moteur de template". C'est un outil qui permet de simplifer la syntaxe au niveau de la structure HTML.

    Il existe plusieurs types de moteurs de templat : Pug, Handlebars, et Mustache.

    Je vais vous parler plus précisément du template Pug anciennement appelé Jade.

    PUG

    Pug nous permet de générer du code HTML, nous pouvons injecter du code JS dans les fichiers pug.

    Il y a aussi des inconvénients si on utilise pug, tout d'abord si on utilise mal les indentations, cela peut nous renvoyer un message d'échec.

    Les commandes pour installer pug :

    npm install pug-cli -g
